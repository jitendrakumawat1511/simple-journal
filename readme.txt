== Description ==

Simple Journal is a clean and minimal WordPress block theme built for bloggers, writers, and content creators. The theme supports Full Site Editing (FSE), allowing you to customize the header, footer, templates, and global styles using the WordPress block editor.

The theme includes a hero section on the homepage and a latest posts section using the Query Loop block. It is lightweight, responsive, and designed for readability and simplicity.

== Installation ==

1. Go to Appearance > Themes in the WordPress admin.
2. Click on "Add New" and then "Upload Theme".
3. Upload the simple-journal.zip file and click "Install Now".
4. Click "Activate" to activate the theme.
5. Go to Appearance > Editor to customize the theme.

Alternatively, you can install the theme manually:

1. Download and unzip the theme package.
2. Upload the `simple-journal` folder to the `/wp-content/themes/` directory.
3. Go to Appearance > Themes and activate the theme.

== Frequently Asked Questions ==

= Does this theme support Full Site Editing (FSE)? =

Yes. Simple Journal is a block theme that fully supports Full Site Editing. You can edit templates, template parts, and global styles from the Site Editor.

= Is this theme responsive? =

Yes, the theme is fully responsive and works on desktop, tablet, and mobile devices.
