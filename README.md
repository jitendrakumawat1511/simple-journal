# Simple Journal WordPress Theme

Simple Journal is a clean and minimal WordPress block theme designed for bloggers and writers. The theme supports Full Site Editing (FSE) and allows users to customize all parts of the website using the block editor.

## Features
- Full Site Editing support
- Responsive design
- Blog-focused layout
- Custom header and footer
- Query Loop for latest posts
- Minimal and clean design
- Gutenberg compatible

## Installation
1. Download the theme zip file.
2. Go to WordPress Dashboard → Appearance → Themes.
3. Click on “Add New” → Upload Theme.
4. Upload the zip file and click Install.
5. Activate the theme.
6. Go to Appearance → Editor to customize the theme.
